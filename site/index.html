Install

Ship in one line

Terminal UI — copy any block. Needs Python 3.11+ and ffmpeg.

zsh · install
$ uv tool install 'vidmcp>=1.1.0'
$ vidmcp --doctor
OK ffmpeg · MCP · creator_path ready
pip · creator extras
$ pip install 'vidmcp[creator]'
# + faster-whisper + mediapipe
mcp · env
$ uvx vidmcp
# VIDMCP_SAM_BACKEND=mock
# VIDMCP_WORKSPACE_ROOT=~/vidmcp-workspaces

One-shot polish

MCP tool run_talking_head_polish — orient → denoise → BGM → captions → export.

Apple Silicon matte

pip install 'vidmcp[mlx]' then VIDMCP_SAM_BACKEND=mlx.
