02
Product

Built for agents, not dashboards

Audio

Denoise + BGM

afftdn · compressor · loudnorm. Ambient pad ducked under speech so laptop speakers still hear music.

Orient

Portrait-safe

Phone rotation (−90°) baked on import. Export 9:16 / 16:9 / 1:1 without stretch.

Matte

Behind subject

SAM / MLX / MediaPipe cutouts. Effects and plates composite under the person.

Captions

Speech burn-in

Whisper word timelines + brand ASS styles. Infographic cards from keywords.

Edit

Smart cuts

Drop dead air and long fillers (um, basically…) from the transcript.

MCP

~89 tools

Projects, recipes, harness gates, education lessons — pack-aware for agents.

03
Flow

From import to export

  1. import_video — bake orientation
  2. process_audio + mix_bgm
  3. replace_background or segment + effects
  4. transcribe_and_caption
  5. export_render — reels / youtube / square
04
Install

Ship in one line

shell 
uv tool install 'vidmcp>=1.1.0'
vidmcp --doctor
shell extras 
pip install 'vidmcp[creator]'  # Whisper + MediaPipe

MCP: point your host at uvx vidmcp with VIDMCP_SAM_BACKEND=mock for offline demos.
