Install

Ship in one line

Needs Python 3.11+ and ffmpeg. Copy any block.

zsh · install
$ uv tool install vidmcp
$ vidmcp --doctor
OK ffmpeg · server · tools
pip · extras
$ pip install 'vidmcp[creator]'
# Whisper + MediaPipe for captions & fast matte
mcp · host
$ uvx vidmcp
# VIDMCP_SAM_BACKEND=mock
# VIDMCP_WORKSPACE_ROOT=~/vidmcp-workspaces

One-shot polish

Tool run_talking_head_polish — full pipeline without juggling 20 calls.

Apple Silicon

pip install 'vidmcp[mlx]' then VIDMCP_SAM_BACKEND=mlx.
